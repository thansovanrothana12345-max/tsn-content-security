# User Manual Guide

This manual describes how to navigate, manage cases, and run scan matches.

---

## 1. Login to the Dashboard
1.  Launch the application desktop shortcut.
2.  Input your email/username and password. (Default credentials: `admin` / `AdminPassword123`).
3.  Click **Login**.

---

## 2. Creating a Copyright Case Folder
1.  On the sidebar navigation, select **Cases**.
2.  Click the **Create New Case** button.
3.  Input the Case Title (e.g. "Summer Revealer Campaign") and description.
4.  Click **Submit** to open the case dashboard.

---

## 3. Uploading Original Reference Videos
1.  Inside the case details view, go to the **Original Videos** tab.
2.  Drag and drop the video files (MP4 or MKV formats, max size 2GB).
3.  The file is uploaded in chunks, and a background fingerprinting task is enqueued.
4.  When processing finishes, the status updates to "Hashed".

---

## 4. Crawling & Scanning Links
1.  Go to the **Evidence Scanner** tab.
2.  Paste the target URL (e.g., a YouTube or TikTok video link) into the scan input.
3.  Click **Trigger Scan**. The task is enqueued in the background.
4.  Once completed, the matched similarity score, screenshot comparisons, and platforms metadata are logged in the evidence list.
